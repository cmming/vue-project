# vue-project

## 项目使用技术：vue2.0+vue-router2.0+axios+vuex

## 项目优点：

* 页面的ui部分实现公共部分的复用。

* 使用vue-router 异步加载view，实现异步加载，同时提高项目的扩展性。同时控制系统的内部的各种正常跳转，并且统一处理异常跳转。

* 封装axiost，对系统的所有请求进行一些统一的处理，让系统的请求结构更加分明。
